
# SuperTwister – Service Manual

## Service Login
- Debug → Service Login
- Passwort erforderlich

## PID Tuning
- PID Panel öffnen
- Kp zuerst, dann Ki, dann Kd einstellen
- Änderungen beobachten im RPM-Graph

## Encoder
- PPR korrekt einstellen
- Signal sauber (keine Preller)

## Wartung
- Mechanik regelmäßig prüfen
- Encoder & Kabel inspizieren
