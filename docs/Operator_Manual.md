
# SuperTwister – Operator Manual

## Start
- Maschine einschalten
- Sicherheitsstatus prüfen (grün)
- Rezept wählen oder Parameter einstellen
- START drücken

## Stop
- STOP drücken
- Not-Aus nur im Gefahrenfall

## Anzeigen
- RPM: Ziel-Drehzahl
- Counter: verbleibende Umdrehungen
- State: IDLE / RUNNING / SAFE

## Sicherheit
- Tür offen oder Not-Aus -> SAFE
- Neustart erst nach Behebung möglich
